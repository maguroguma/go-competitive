## 小さい方からK番目の値がXである、の言い換え

**`(X-1)` 以下の数は `K` 個未満しかないが、 `X` 以下の数は `K` 個以上ある。**

例:

`1, 1, 2, 2, 2, 2, 2, 4, 4`

小さい方から `7` 番目は `2`

- 1以下の数は `2` 個 `<` `7` 個
- 2以下の数は `7` 個

別の言い方をすると、

**「`X` 以下の数が `K` 個以上あるような最小の `X`」が小さい方から `K` 番目の数。**

ここまで読み替えられると、2分探索が十分に想起できる。
