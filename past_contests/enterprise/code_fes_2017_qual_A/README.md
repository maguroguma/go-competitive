## A問題（@2020-03-30）

与えられた文字列が `YAKI` から始まるかどうかを判定する問題。

正規表現を使ってみた。

Goの正規表現は遅いと言われているが、一部のめんどくさい問題を瞬殺できる可能性があるので、できるだけ使っていきたい。
