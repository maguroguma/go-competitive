  - 色々な解法があるので、とりあえず全部試してみてどれがベストなのか反省したい。

- Eは組み合わせを使った数え上げに帰着する問題。
  - 全体を見るのではなく、個別の要素ごとに考えるのは典型手法。
- Fは「最小包含円」という、幾何の分野では超典型問題。
  - 幾何はちょっと集中特訓する気にはならないが、出たものはちゃんと理解していきたい。
