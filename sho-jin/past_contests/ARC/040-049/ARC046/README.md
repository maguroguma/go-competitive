## B問題（@2020-02-14）

苦手なゲーム系問題。。の中ではシンプルで簡単そう。
でもちょっと考えてもよくわからない。。

と思ったら、やっぱり整理すると結構簡単だった。

**逆から考えるのが良い。**

まずは `a == b` の状況を石が0個の状態から考えてみると、必勝の状況がわかってくる。
思考に慣れてきたら、 `a > b` や `a < b` についても考えてみると、冷静に詰めることができる。

**典型: ゲーム系は逆から考えるのが定石**
