## C問題

これも結構難しいと思うが。。

以下の手順でOKのはず。

1. 一番左を0にする
2. 区間は `n` にして、一番左意外に正の数を加算して、 `n-1` の倍数となるようにする。
3. 一番左を除いた `n-1` の区間を選択し、現在のそれぞれの値の正負逆転した数を加算する。

2については、 `n` を1回加算すると `n-1` のあまりが1増えるので、それを利用すれば良い。
