## 1桁の整数の文字列変換について

`'1' - '0' == 1` であることから `'0'` を右に移項して `i + '0'` で変換できる。
ただし、変数 `i` がint型だとコンパイラに怒られるので、
`rune(SS[i]) + '0'` のようにrune型でキャストしてやること！
