## C問題（@2020-03-22）

中間の街について全探索をして、ダイクストラを行えば良い。
制約が結構きつめなので、無駄なダイクストラは行わないように注意すべき。

実はグラフは連結とは限らないため、それが微妙にコーナーケースになっている。

今回の問題に関しては、問題文が若干不親切な感じがするが、
**ダイクストラの結果が無限大、すなわち到達不能という状況を示すことは、今一度再認識しておいたほうが良さそう。**
