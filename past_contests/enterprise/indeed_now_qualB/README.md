## D問題（@2020-03-24）

数列の区間をテーマにした数え上げ問題。
Zero Sum Ranges系かと思ったが、ちょっと違う。

余事象的に考える。
つまり、「1つ以上含む」というのを重複なく数え上げるのは難しいので、
「全く含まれない」というものを簡単に数えて、それを全体から引く、ということを行えば良い。
