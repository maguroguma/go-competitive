### 解説放送を観て

考え方が勉強になる。

- `(1, 2) = (0, 1) + (1, 1)` のように分解して考えると、最終的に数Aの格子点の問題に帰着できる
- 移動量の総和に注目すると、どちらも `3` 移動することになる
  - ここから `X+Y` が総移動量と断定でき `移動回数 = (X+Y) / 3` が自然に導ける。
