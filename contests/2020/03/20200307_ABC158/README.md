## E問題の反省を雑多に

**忘れた頃に解き直したい。**

- 区間の数値の抜き出し方を頭に入れておきたい
  - `[i, j]` の数値は `(D[i] - D[j]) / 10^(桁数)`
    - `D[i]`: 下から `i` 桁目までを数値として評価したもの。
- **「剰余を求める」というのは、かなり無茶な桁数で要求されても意外となんとかなる！というのを強く意識したい。**
  - Digit paradeとかでもそこがネックになっていた。
