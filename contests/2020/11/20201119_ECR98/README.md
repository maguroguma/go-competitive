## D問題

確率を効かれているが、結局は「長さ `n` の区間を奇数長のいくつかの区間に分割する方法の数」がわかれば良い。  
以下のDPで求まる。

`dp[i] := i番目まで決めたときの通り数`

こうすると `dp[i] := dp[i-1] + dp[i-3] + dp[i-5] + ..` のようになる。  
愚直にやると `O(N^2)` だが、偶奇ごとに1つとびの累積和を保持しながらDPをすると、遷移が `O(1)` となる。
