$$|x|+|y| = \max(x+y, x-y, -x+y, -x-y)$$
$$|x|+|y|+|z| = \max(x+y+z, x+y-z, x-y+z, x-y-z, -x+y+z, -x+y-z, -x-y+z, -x-y-z)$$
