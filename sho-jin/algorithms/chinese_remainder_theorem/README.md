# 中国の剰余定理

## 2元の場合

`n1, n2` が互いに素なとき（ `gcd(n1, 2) == 1` のとき）、

連立合同式 `x = a1 mod n1, x = a2 mod n2` を満たす `x` が
**`[0, n1*n2)` の範囲にただ1つ存在する。**

[2元の場合の証明](https://mathtrain.jp/chinese)は「高校数学の美しい物語」さんの記事がわかりやすい。
