## （解説放送より）B問題の具体的なコード

```cpp
int main(void) {
  int N;
  cin >> N;
  for (int x = 111; x <= 999; x += 111) {
    if (x >= N) {
      cout << x << endl;
      return 0;
    }
  }
}
```

111で割る、とかいう方法もある（`ceil(x/111)*111`）。
（誤差には要注意。）
